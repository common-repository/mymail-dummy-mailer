== Description ==

> This Plugin requires [MyMail Newsletter Plugin for WordPress](https://evp.to/mymail?utm_campaign=wporg&utm_source=MyMail+Dummy+Mailer)

A Dummy Mailer for the [MyMail Newsletter Plugin for WordPress](https://evp.to/mymail?utm_campaign=wporg&utm_source=MyMail+Dummy+Mailer). Requires at least version 2 of the MyMail Plugin

== Installation ==

1. Upload the entire `mymail-dummy-mailer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings => Newsletter => Delivery and select the `DummyMailer` tab
